Köra Python

------Ditt första program

För att kunna programmera i Python behöver man Python installerat på sin dator och ett program där man skriver och kör koden ifrån (ett så kallat IDE).

Det finns däremot andra alternativ som gör det möjligt att man kan programmera i Python online i ett molnbaserat system. Fördelen med att programmera online är att man börjar koda med en gång utan att installera och konfigurera programmet IDE. Idag finns det massor av sådana online system man kan välja. Om man skulle nämna några så är det Google Colab och Replit.

I den här kursen ska vi progammera i Replit. Det är ett simidigt molnsystem, där blir igång direkt efter ha skapat ett konto.

På Replit kan man även koda utan att skapa ett konto. Men om du vill spara ditt arbete så krävs det att du skapar ett användarkonto på Replit. Jag rekommenderar att du skapar ett användarkonto på Replit (ett användarkonto på Replit kostar inget men man får begränsad lagring på 500MB)

Klick på följande länk som tar dig till Replit sidan.
